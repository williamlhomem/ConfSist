Resolução¶

Pede-se, primeiramente, para se criar um vetor de entrada contendo 100 observaes, geradas por meio de uma distribuio normal N(0,1), supondo que elas pudessem ser oriundas de um sensor qualquer, colhidas a 100 hz durante 1 segundo. Alm disso, tambm foi pedido para se criar um termo estocstico com N(0;0,25).

np.random.seed(42)
x = np.random.normal(0, 1, size=(100,))

# criao do termo estocstico
e = np.random.normal(0,.25, size=(100,))

Aps a criao desses dois vetores, pode-se determinar, por meio da equao abaixo, um conjunto de sada que possua uma boa correlao linear com o vetor de entrada. Note que, como $y$ ser computado por meio de dois vetores de tamanho 100, esse novo vetor tambm ter tamanho 100. Ademais, pode-se observar que a relao possui um termo linear ($\beta _0$) igual a 3 e um termo ngular ($\beta _1$) igual a 2. Verifica-se, por meio do scatter plot abaixo, que as variveis criadas artificialmente apresentam a correlao linear esperada.

Criados os conjuntos de entrada e sada, pode-se realizar uma regresso linear por mnimos quadrados para o ajuste de $x$ em $y$. Observa-se que a curva ajustada apresenta erros absoulutos entre os $\beta _0$ de 0,002 e entre os $\beta _1$ de 0,046, garantindo assim uma boa representao, com um $R^2$ de 0,9997.

In [4]:
# reta real, sem rudo
yt = 3 + 2*x

# criao e treinamento do regressor
lreg = LinearRegression()
lreg.fit(x.reshape(-1,1), y)

# predio de x
y_ = lreg.predict(x.reshape(-1,1))
rsquared = np.round(r2_score(yt, y_), decimals=4)

# coeficientes da reta
w = np.round(lreg.coef_[0], decimals=3)
b = np.round(lreg.intercept_, decimals=3)

# plotagem
fig, ax = plt.subplots(figsize=(10,10))
ax.plot(x, yt, 'r', label=r'Valor real: $ \beta _0$: 3 e $\beta _1$: 2')
ax.plot(x, y_, 'g', label=r'Valor predito: $ \beta _0$: {} e $\beta _1$: {} ($R^2$: {})'.format(b, w, rsquared))
ax.scatter(x, y)
ax.set_xlabel(f'$x$', size=15)
ax.set_ylabel(f'$y$', size=15)
ax.grid()
ax.legend(prop={'size': 15})

Como forma de experimentao, pode-se realizar uma regresso quadrtica por mnimos quadrados como forma de verificao se o conjunto de dados trabalhado se adequa melhor a uma curva de grau 2. Para tanto, basta criar um vetor $x^2$ e alimentar o modelo de regresso linear com os vetores $x$ e $x^2$. Observe que, mesmo que os coeficientes de determinao tenham dado muito parecidos, o suposto polinmio de segundo grau possui um coeficiente $\beta_2$ praticamente nulo, fazendo com que a curva se assemelhe, em muito, com uma reta.

In [6]:
# criao do vetor de segunda ordem e adequao do conjunto de entrada
x_squared = x**2
xtrain = np.c_[x, x_squared]

# criao e treinamento do regressor
lreg = LinearRegression()
lreg.fit(xtrain, y)

# predio de x
y_ = lreg.predict(xtrain)
rsquared = np.round(r2_score(yt, y_), decimals=4)

# coeficientes da curva
w = np.round(lreg.coef_, decimals=3)
b = np.round(lreg.intercept_, decimals=3)

# plotagem
x_sorted = x.copy()
x_sorted.sort()
y_sorted = y_.copy()
y_sorted.sort()

fig, ax = plt.subplots(figsize=(10,10))
ax.plot(x_sorted, y_sorted, 'g', label=r'$ \beta _0$: {}, $\beta _1$: {} e $\beta_2$: {} ($R^2$: {})'.format(b, w[0], w[1], rsquared))
ax.scatter(x, y)
ax.set_xlabel(f'$x$', size=15)
ax.set_ylabel(f'$y$', size=15)
ax.grid()
ax.legend(prop={'size': 15})

Por ltimo, ainda em carter experimental, pode-se verificar como se comportaria a regresso caso o termo estocstico possusse uma distribuio com menor ou maior variabilidade. Nos grficos abaixo, observa-se regresses com gerao de rudo para desvios padro 5 vezes menor e 5 vezes maior, respectivamente, do que o que foi apresentado acima. Note como o termo de erro impacta diretamente na disperso dos pontos em relao reta verdadeira ($y = 3 + 2x$) e na performance da regresso linear. Pode-se ver, tambm, que embora os grficos de resduos no mostrem tendncia ou padro, as amplitudes dos erros absolutos apresentam valores maiores com o aumento da disperso do termo estocstico.
